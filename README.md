### 遊戲操作說明
```text

↑:順轉方塊 Clockwise
↓:下移方塊 Down
←:左移方塊 Left
→:右移方塊 Right
space:快速掉落方塊至底 Quick down
enter:暫停/繼續遊戲 Game stop/Game resume
```
